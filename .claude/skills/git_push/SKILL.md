name: git_push
description: 변경사항을 스테이징, 커밋 메시지 작성 후 원격 저장소에 푸시합니다
---

다음 단계를 순서대로 실행하고, 각 단계에서 에러가 발생하면 **즉시 실행을 중지**하고 에러 메시지를 사용자에게 보여주세요:

1. `git add .`를 실행합니다. 에러 발생 시 즉시 중단.
2. `git status`를 실행하여 변경 사항을 확인합니다. 에러 발생 시 즉시 중단.
3. 적절한 커밋 메시지를 생성합니다. (커밋할 변경 사항이 없다면 여기서 중단하고 알림)
4. `git commit -m "[메시지]"`를 실행합니다. 에러 발생 시 즉시 중단.
5. `git push`를 실행합니다. 에러 발생 시 즉시 중단.

작업 완료 시 결과를 요약해서 알려주세요.
